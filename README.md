## Features
* **Complete Database:** Includes all Balls and Passives (including the **Regal Update**).
* **Smart Evolution Tracker:** Automatically calculates available evolutions based on your inventory.
* **Inventory Management:** Drag & drop interface to replicate your current in-game run.
* **Target Planning:** Click on any item to "target" it and see exactly which ingredients you are missing.
* **Various Surprises:** Stuff I built in but forgot to list.
